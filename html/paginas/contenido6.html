Tipos de datos primitivos

number

Engloba cualquier tipo de número, ya sea positivo, negativo o con decimales. Podemos realizar cualquier tipo de operación básica como sumar, restar, multiplicar o dividir, ademas de poder compararlos

También podemos hacer uso de otras herramientas como:

typeof variable Devuelve el tipo de variable
isNaN(variable) Verdadero si la variable parámetro no es un número
variable.toString() Devuelve la variable como cadena
variable.toFixed() Devuelve la variable redondeada a tanto decimales cono indiquemos

String

Es cualquier cadena de caracteres o texto. Podemos concatenar cadenas, compararlas…

Para concatenar cadenas nos vale con hacer ‘variable1’ + ‘variable2’

Otras herramientas son:

typeof variable Igual que con los números, devuelve el tipo de variable
variable.lenght Indica la cantidad de caracteres que contiene la cadena
variable.includes(‘cadena’) Verdadero si la variable (tipo texto) contiene la cadena especificada
variable.slice(inicio, final) Devuelve una cadena entre las dos cifras indicadas
variable.replace(esto, esto_otro) Intercambia el contenido de una cadena con otro
variable.trim() Limpia los espacios de una cadena que se entren antes o después
variable.split(‘separador’) Devuelve un array con las cadenas separadas por el separador

Boolean

Tipo de dato que almacena el estado de una variable. Solo puede ser true o false

Para hacer uso de este tipo de dato, usamos Boolean(expresion). Lo cual nos devolverá el resultado

Otros tipos de datos

También existe otro tipo de datos los cuales se utilizan para indicar la ausencia de valor, estos son undefined y null

undefined es el valor que nos devolverá una variable que hemos declarado en algún punto del ámbito actual pero que no se le ha asignado un valor

Por otro lado null es el valor que nos devolverá una variable sin un valor establecido. Debemos asignarle a una variable el valor null para que esta no tenga valor. También hay que decir que se le asignamos a una variable el valor null, el typeof de esta variable nos devolverá object
