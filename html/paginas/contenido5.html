Variables

¿Que tipos de variables hay?

Podemos crear variables con:

  • var varible1 = ‘var1’
  • Pueden usarse incluso si no se les ha asignado un valor
  • let variable2 = ‘var2
  • Tienen que tener un valor asignado para poder usarlas
  • const variable3 = ‘var3’
  • Conservan su valor durante todo el programa

Para recuperarlas no tenemos nada mas que escribir su identificador en la consola o donde nos haga falta

  • Ejemplo:
  • console.log(variable1)

Las variables pueden ser globales o locales. Se diferencias por donde podemos llamar a cada una de ellas. Mientras que las variables globales serán accesibles desde cualquier parte del programa, las variables locales solo serán visibles desde la parte del código que se ejecutan
