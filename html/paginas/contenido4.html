Probandolo en la consola

Para hacer unas pruebas vamos a abir la consola de nuestro navegador, que normalmente se abre pulsando F12

Vamos a probar diferentes maneras de mostrar mensajes con:
alert(‘Prueba’) Mensaje emergente
console.info(‘Prueba’) Información en consola
console.log(‘Prueba’) Log en consola
console.warn(‘Prueba’) Advertencia en consola
console.error(‘Prueba’) Error en consola

También podemos hacer uso de debugger para parar las instrucciones en un punto específico. Esto se suele usar a la hora de depurar el script
